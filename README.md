## goBlog

goBlog是一个前后端分离的通用博客系统，后端使用gin框架开发，遵循Restful接口规范，数据库存储使用mongodb。

[前端仓库](https://github.com/masterZSH/vue-blog)



## 使用
1. 配置文件 修改configs/app.default.ini为/configs/app.ini

2. 调试  go run main.go

3. 构建  go build
